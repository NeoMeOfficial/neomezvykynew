# Reference visuals — trial batch

each trial-batch screen. They're for quick visual orientation only — for full
fidelity, open `handoff/00-prototype/NeoMe-Final-Review.html` (single
self-contained file, works offline) and navigate to the corresponding
artboard ID.

## All shots are 402×540

Native iPhone 14 Pro CSS width is 402px. Heights vary per actual content.
The status bar at top is the iOS status bar (rendered by the prototype shell);
your real app provides its own.
